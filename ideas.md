## 最终选择: 技术极简主义 (Technical Minimalism)

选择理由:
1. **专业性**: 路由器管理界面是专业工具,需要高信息密度和清晰的视觉层级
2. **可读性**: 纯白背景+深色文字保证长时间使用不疲劳
3. **效率**: 最小化动画和装饰,聚焦于功能和数据展示
4. **可扩展性**: 简洁的设计系统易于维护和扩展新功能
5. **符合用户预期**: 参考Ubiquiti UniFi的专业风格,用户熟悉度高

## 设计细节

### 色彩系统
```css
/* 主色调 */
--background: #ffffff
--foreground: #1a1d23
--primary: #0066ff
--primary-foreground: #ffffff

/* 功能色 */
--success: #00c853
--warning: #ffc107
--error: #ff3d00
--info: #00bcd4

/* 中性色 */
--muted: #f5f7fa
--muted-foreground: #6b7280
--border: #e5e7eb
```

### 排版系统
- 标题: IBM Plex Sans Medium (500)
- 正文: IBM Plex Sans Regular (400)
- 代码: IBM Plex Mono Regular (400)
- 字号: 32px/24px/18px/14px/12px
- 行高: 1.5倍字号

### 布局结构
1. **侧边栏**: 60px图标栏 + 200px展开详情
2. **顶部栏**: 64px固定高度,包含搜索、通知、用户信息
3. **主内容区**: 12列网格,24px间距
4. **卡片**: 8px圆角,1px边框,无阴影

### 组件设计
1. **按钮**: 8px圆角,36px高度,16px左右内边距
2. **输入框**: 8px圆角,40px高度,1px边框
3. **表格**: 斑马纹效果,悬停高亮行
4. **图表**: 使用Recharts,蓝色系配色
5. **图标**: Lucide Icons,24px尺寸

### 页面结构
1. **仪表盘**: 网格布局,显示系统概览
2. **网络管理**: 表格+详情面板
3. **容器管理**: 卡片网格布局
4. **虚拟机管理**: 列表+操作按钮
5. **硬件监控**: 实时图表+数据表格
6. **系统设置**: 表单布局
